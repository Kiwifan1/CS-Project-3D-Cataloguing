To install and run my application:
 1. Simply clone from github
 2. Install MySQL connector to java from ![Here][https://dev.mysql.com/downloads/connector/j/]
 3. Add/Change the config.properties file in 'Logic' Folder to match your credentials
 4. Change file path in ConnectLogic.java to read config.properties file
 5. Run Item.sql on your SQL connection
 6. Run program, clear out files as needed
